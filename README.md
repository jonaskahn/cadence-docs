# Cadence docs

Documentation for **Cadence**, built with [Starlight](https://starlight.astro.build/) (Astro). Content lives in `src/content/docs/`; navigation is configured in `astro.config.mjs`.

## Requirements

- [Node.js](https://nodejs.org/) 22.12 or newer

## Quick start

```bash
cd docs
npm install
npm run dev
```

Open the local URL shown in the terminal (typically `http://localhost:4321`).

## Build

```bash
npm run build
npm run preview
```

`preview` serves the static output in `dist/` for a local production check.
